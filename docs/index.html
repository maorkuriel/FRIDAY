How It Works

Your PM workflow, fully automated

F.R.I.D.A.Y reads your product context, routes your requests to the right skill, and writes everything to a knowledge base that gets smarter with every use.

Start with context, not templates

Set up your knowledge base once. Every skill reads from it automatically — your OKRs shape your prioritization, your strategy informs your PRDs, your retros feed your next sprint.

  • 6-question personalization wizard
  • Auto-researches your company from a URL
  • 21+ knowledge directories that compound over time
$ /pm-setup
→ Reading company context...
→ Setting up knowledge base...
✓ pm-context.md created
✓ okrs.md created
✓ team.md created
F.R.I.D.A.Y ✓ Ready — Snyk Platform

Multi-agent document review

Ask F.R.I.D.A.Y to review your PRD and three expert subagents fire simultaneously — an engineer, an executive, and a user researcher — each with a risk-rated critique.

  • Parallel reviews in seconds
  • Each subagent has a distinct lens and priority
  • No more "looks good to me" without substance
$ review this PRD
Spawning 3 subagents...
🔧 Backend Engineer
Risk: API rate limits not addressed
📊 Executive
Risk: Success metric lacks baseline
👤 User Researcher
Risk: 0 user validation cited

Intelligent routing

You don't need to remember 39 commands. Describe what you need in plain language — F.R.I.D.A.Y routes to the right skill, reads the right files, and delivers output in your voice.

  • "prep me for my exec meeting" → /meeting-prep
  • "customers are churning" → /retention-engagement
  • "should we build X?" → /opportunity-assessment
$ our churn is up this quarter
→ Routing to retention-engagement
→ Reading metrics/...
→ Reading okrs.md...

from 62% → 54% since the redesign.

Install

Up and running in 3 minutes

Requires Claude Code and a Claude Pro, Team, or Enterprise subscription.

# Option 1: Plugin marketplace (recommended)
claude plugin marketplace add maorkuriel/friday
claude plugin install friday
# Option 2: Manual install
git clone https://github.com/maorkuriel/FRIDAY.git
cd FRIDAY && ./install.sh
# Then set up your knowledge base
/pm-setup
